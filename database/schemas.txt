message ([id], content, user, conv-id)
  -> un message d'une conversation
  id : idenfiant unique du message
  content : contenu du message chiffré avec la clé partagée de la conversation
  user : l'utilisateur auteur du message (chiffré ?)
  conv-id : la conversation à laquelle partient le message

convs (conv-id, name, …)
  -> les conversations auxquelles un utilisateur participe
  conv-id : identifiant unique de conversation
  name : nom de la conversation
  … : autre informations (sujet, image…)

user ([user-id], name, password-hash, {public-key}, …)
  -> les propriétés d'un utilisateur
  user-id : identifiant unique
  name : nom d'utilisateur
  password-hash : le mot de passe hashé de l'utilisateur
  public-key : clé publique de l'utilisateur
    -> si non unique, alerte
  … : autre informations (avatar, status, message perso…)

keys ([key-id], conv-id, user-id, shared-key)
  -> les conversations auxquelles un utilisateur participe
  -> avec la clé utilisée, chiffrée par la clé publique de l'utilisateur
  key-id : identifiant unique
  conv-id : identifiant de conversation ;; référence user
  user-id : identifiant de l'utilisateur
    -> un utilisateur peut être enregistré plusieurs fois sur une conversation
    -> s'il a plusieurs clés publiques
  shared-key : la clé chiffrée avec la clé publique de l'utilisateur
